# 🏨 Hotel Showcase Website

A responsive hotel website built with **HTML, CSS, JavaScript, and PHP**. Designed to showcase rooms, facilities, and services for a hotel.

---

## 📌 Features

- Responsive design for desktop, tablet, and mobile  
- Home page with hero banner, featured rooms, and services  
- Rooms listing with descriptions and images  
- Contact form using PHP for message submission  
- Navigation bar with smooth scrolling to sections

## 🚀 How to Run Locally
1. Clone the repository:
```bash
git clone https://github.com/hzC4nfly/hotel-showcase.git
```
2. Open index.html in your browser (or use a local server like XAMPP for PHP functionality)
3. Test contact form via PHP (requires server environment)
